\section{Methodology}

Under this section the methodology of the project will be described in detail. The
stages of development from start to finish will be explained including strategies, design
choices, development lifecycles, and technological specifics.

The main goal of this project is to promote citizen science by building a platform
with an integrated machine learning model for birdwatchers and enthusiasts. The project
will follow a structured methodology that combines a comparative analysis of different
machine learning models and applied software development.

This section describes the methodology that was followed when developing the project. 
The project is divided into two seperate phases. The two phases are machine learning
phase of the project and web platform development part.

Machine learning phase includes algorithm selection, data collection, model development, and evaluation.
On the other hand, web platform development phase includes design and development of the website, integration of the machine learning model, and testing.
