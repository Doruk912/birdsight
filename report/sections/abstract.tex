\section*{\textbf{ABSTRACT}}
\addcontentsline{toc}{section}{ABSTRACT}

Birds are a vital part of ecosystems across the globe. 
They help keep the balance in the ecosystem and act as strong bioindicators of ecosystem health. 
This is why it is important to monitor bird populations on a large scale. 
However, traditional recording methods depend on manual observations and expert knowledge.
This makes large scale monitoring very difficult to scale and time consuming.
This project presents BirdSight, a web-based citizen science platform built using Java Spring Boot for the backend and Next.js for the frontend. 
The platform also integrates a machine learning model for species identification. The model uses deep learning techniques to identify the bird species from input images.
Multiple convolutional neural network architectures, including VGG16, ResNet50, InceptionV3, MobileNetV2, and DenseNet121, were trained, evaluated, and compared.
The best performing and most suitable model for deployment was further optimized and incorporated into the platform.
The platform aims to support large-scale bird monitoring, promote citizen science, and encourage the public to engage with biodiversity around them.
The platform allows users to upload observations, view sightings on an interactive map, join a community of like-minded people, and help contribute to biodiversity data collection.

\noindent\textbf{Keywords:} Bird Species Classification, Citizen Science, Convolutional Neural Networks, Deep Learning, Image Classification, Biodiversity Monitoring, Web Platform, Comparative Analysis
